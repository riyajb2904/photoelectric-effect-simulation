## **Photoelectric Effect Simulation**



An interactive Python simulation of the **Photoelectric Effect** built using **Pygame**.



This project visually demonstrates Einstein's Photoelectric Effect by simulating photon interactions with different metals and showing electron emission based on wavelength, intensity, work function, and stopping voltage.



**## Features**



\* Interactive wavelength slider (200–800 nm)

\* Adjustable light intensity

\* Adjustable stopping voltage

\* Multiple metal selections:



&#x20; \* Sodium

&#x20; \* Potassium

&#x20; \* Zinc

&#x20; \* Copper

&#x20; \* Gold



\* Animated photon beams

\* Electron emission visualization

\* Real-time calculations of:



&#x20; \* Photon Energy

&#x20; \* Work Function

&#x20; \* Threshold Frequency

&#x20; \* Maximum Kinetic Energy

&#x20; \* Photoelectric Current



\* Color-coded wavelength representation

\* Emission and non-emission states



**## Physics Concepts Demonstrated**



The simulation is based on Einstein's Photoelectric Equation:



KEmax = hf − φ



Where:



\* h = Planck's Constant

\* f = Frequency of Incident Light

\* φ = Work Function of the Metal



Electron emission occurs only when the photon energy exceeds the work function of the selected metal.



**## Controls**



| Control                 | Function                             |

| ----------------------- | ------------------------------------ |

| Wavelength Slider       | Changes wavelength of incident light |

| Intensity Slider        | Changes light intensity              |

| Stopping Voltage Slider | Applies opposing voltage             |

| Left Arrow Key          | Select previous metal                |

| Right Arrow Key         | Select next metal                    |
